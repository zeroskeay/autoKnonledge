# AI 自适应学习助手

AI 驱动的个性化学习路径引擎。通过对话式交互，自动诊断你的知识水平，生成定制化学习路线图，并提供精准匹配当前层级的学习材料。

## 特性

- 🧠 **自适应诊断** — AI 通过串行问答，自动构建知识依赖树并定位你的当前层级
- 📚 **深度学习引擎** — 基于诊断报告，生成严格匹配当前 Level 的教材和练习
- 🔄 **三大 AI 提供商** — 支持 Groq、Gemini (Google)、GLM (智谱)
- ⚡ **超快响应** — 默认使用 Groq Llama 3.3，体验极致速度
- 🧠 **双模引擎** — 架构师模式 (Architect) + 深度学习模式 (Engine)
- � **会话持久化** — 自动保存进度，支持历史回溯
- �💬 **流式输出** — 逐字显示 AI 响应，体验更流畅
- 📝 **Markdown 渲染** — 自动渲染代码块、列表、表格等结构化内容
- 🔑 **前端配置** — API Key 仅存储在本地浏览器 localStorage 中

## 🛠️ 快速开始

1. **获取 API Key**:
    - **Groq**: [console.groq.com](https://console.groq.com/keys) (推荐，速度极快)
    - **Gemini**: [aistudio.google.com](https://aistudio.google.com/apikey) (免费且强大)
    - **GLM**: [open.bigmodel.cn](https://open.bigmodel.cn) (国产之光)

### 2. 打开应用

直接用浏览器打开 `index.html` 即可，无需安装任何依赖。

```
# 方式一：直接双击打开
index.html

# 方式二：使用本地服务器（推荐，避免 CORS 问题）
npx serve .
# 或
python -m http.server 8080
```

### 3. 配置 API

1. 选择 AI 提供商（Grok / Gemini / GLM）
2. 输入对应的 API Key
3. 等待模型列表自动加载
4. 选择模型后点击 **「开始学习对话」**

### 4. 开始学习

应用分为两个阶段：

#### 阶段一：自适应诊断（架构师模式）

AI 会依次询问：
1. **学习主题** — 你想学什么？
2. **学习目的** — 解决问题 / 面试 / 兴趣 / 学术？
3. **已有背景** — 你目前掌握了什么？
4. **探针测试** — 通过问答定位你的精确层级

诊断完成后，AI 输出包含以下内容的 **最终报告**：
- 学习契约概览（主题、目的、定位结论）
- 学习全景路线图（全部层级 + 当前位置 📍）
- 诊断与建议

#### 阶段二：深度学习（执行引擎模式）

点击聊天顶部的 🎓 按钮进入学习模式，AI 会生成：
- **目标校准** — 当前 Level 的学习价值
- **核心信息** — 概念解释 + 关键逻辑 + 适用边界
- **适配性练习** — 匹配当前难度的场景练习
- **经验总结** — 知识串联与前置检查

完成后可继续对话，AI 会持续提供当前层级的学习支持。

### 5. 会话管理

- **自动保存** — 每条消息后自动保存到 localStorage
- **恢复会话** — 返回设置页，点击历史会话即可继续
- **删除会话** — 点击会话右侧的 🗑️ 按钮
